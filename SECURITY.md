# Security Policy

## Supported Versions

Security updates are provided for the current `main` branch and the latest
released deployment of Driveway Estimator Pro.

## Reporting a Vulnerability

Please do not open a public issue for suspected vulnerabilities. Send a private
report to the project owner with:

- Affected route, command, or package
- Steps to reproduce
- Expected impact
- Any logs or screenshots that do not contain secrets

We will acknowledge valid reports as soon as possible, prioritize fixes by
impact, and publish remediation notes after a fix is available.
